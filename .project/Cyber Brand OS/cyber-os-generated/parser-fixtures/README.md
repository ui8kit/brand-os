# Cyber OS Parser Fixtures

These fixtures are derived from `Cyber Brand OS/example.html` and are intended to validate how a parser splits classes into structural, semantic, decorative, and unknown buckets.

## How To Use
1. Read a fixture file from this directory.
2. Classify each class according to the parser contract.
3. Compare the parser result to the `expected` buckets.
4. Emit warnings for any mismatches or unknown classes.
