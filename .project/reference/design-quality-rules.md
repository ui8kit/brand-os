# Design Quality Rules

Anti-slop rules for distinctive, production-grade interfaces. Apply to all brand work.

## Design Thinking

Before building, commit to a clear aesthetic direction:

- **Purpose**: What problem does this interface solve? Who uses it?
- **Tone**: Pick a direction and commit: brutally minimal, warm/organic, retro-futuristic, luxury/refined, playful, editorial/magazine, brutalist/raw, art deco, soft/pastel, industrial/utilitarian
- **Constraints**: Framework, performance, accessibility requirements
- **Differentiation**: What makes this unforgettable? What will someone remember?

Bold maximalism and refined minimalism both work — the key is intentionality, not intensity.

## Typography

- NEVER use Inter, Roboto, Arial, or system fonts as primary choices
- Pick distinctive, characterful fonts that match the brand personality
- Pair a display font with a complementary body font
- Unexpected, characterful font choices elevate everything

Good pairings:
- Fraunces + DM Sans (warm, organic)
- DM Serif Display + Inter (bold, modern)
- Playfair Display + Lora (editorial, sophisticated)
- Libre Baskerville + Source Sans 3 (luxury, refined)
- Outfit + Nunito (playful, friendly)
- Geist + Geist (minimal, precise)

## Color & Theme

- Commit to a cohesive aesthetic with CSS variables for consistency
- Dominant colors with sharp accents outperform timid, evenly-distributed palettes
- Build from emotional direction, not random picker
- Warm: ambers, oranges, rose, warm browns
- Cool: slates, blues, teals, cool grays
- Bold: high saturation accent on low-saturation base

## Spatial Composition

- Unexpected layouts, asymmetry, overlap, diagonal flow, grid-breaking elements
- Generous negative space OR controlled density — both work when intentional
- Match density to purpose: landing pages need breathing room, dashboards need compact information

## Backgrounds & Visual Details

- Create atmosphere and depth rather than defaulting to solid colors
- Contextual effects and textures that match the overall aesthetic
- Gradient meshes, noise textures, geometric patterns, layered transparencies, dramatic shadows, decorative borders, grain overlays

## Motion

- Focus on high-impact moments: one well-orchestrated page load with staggered reveals creates more delight than scattered micro-interactions
- Prioritize CSS-only solutions when possible
- Use scroll-triggering and hover states that surprise
- Match complexity to vision: maximalist designs need elaborate effects, minimalist designs need restraint

## NEVER Do

- Generic purple gradients on white backgrounds
- Overused font families (Inter, Roboto, Arial, system fonts) as primary choices
- Cliched color schemes and predictable layouts
- Cookie-cutter component patterns with no context-specific character
- Same design across different brands — every brand must be unique
- Flat textureless designs for brands that need warmth or atmosphere
- Tech-product styling for non-tech brands
- Decorative motion that doesn't improve comprehension

## The Rule

Every interface should feel genuinely designed for its context. No two brands should converge on the same choices. Vary between light and dark themes, different fonts, different aesthetics. Execute the vision with precision — elegance comes from commitment, not from adding more effects.
